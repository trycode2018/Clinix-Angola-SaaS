# Clinix Angola SaaS

Plataforma SaaS multi-clínica para gestão de consultas médicas, desenvolvida para o contexto angolano e preparada para operação em clínicas, centros médicos e hospitais.

## Principais Funcionalidades

- Gestão de Clínicas
- Gestão de Médicos
- Gestão de Pacientes
- Agendamento de Consultas
- Notificações por Email
- Notificações por WhatsApp
- Dashboard Analítico
- Auditoria e Logs
- Arquitectura Multi-Tenant
- Segurança baseada em JWT
- Deploy com Docker
